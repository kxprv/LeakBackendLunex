## About The Project

This project is a leaked backend system originally developed by **kxprv**. It provides a full-featured server environment focusing on competitive gameplay, seasonal persistence, and a custom economy.

_**Disclaimer:** This project was made for educational purposes and archival interest. If you are **kxprv** and have any issues with this project, please do not hesitate to contact us through a GitHub Issue._

### Key Features
* **Authentication**: Login and connection system for all users.
* **The Arena**: Core logic for matchmaking and combat interactions.
* **Double Shop**: Dual-currency shop system for seasonal items.
* **Full Season Support**: Architecture designed to sustain data throughout a whole season.
* **Database**: Powered by **MongoDB** for flexible data handling.
